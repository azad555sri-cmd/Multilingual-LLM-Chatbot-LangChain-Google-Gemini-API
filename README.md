LLM-Chatbot-LangChain-Google-Gemini-API

A secure, multilingual LLM chatbot built using LangChain and Google Gemini API, leveraging API key authentication to generate real-time AI responses via a Streamlit interface.

Features

- Chat with a Large Language Model (LLM) powered by Google Gemini.

- Multilingual support – translate and respond in multiple languages.

- Real-time responses using Streamlit.

- Secure API integration with Google Gemini API key.

- Easy to extend for additional languages or LLM features.

Tech Stack

- Python

- LangChain – for LLM orchestration

- Google Gemini API – for AI model responses

- Streamlit – for web-based UI

- langchain-google-genai – LangChain provider for Google Gemini
